## 📋 About pulso

pulso offers AI agents that handle tasks for you. It manages multiple agents working together. You can use 12 different messaging channels. pulso can control your computer fully and remembers things using a knowledge graph. It helps automate tasks with ease, even if you don’t know how to code.

pulso works well on Windows. It fits users who want to try personal AI assistants without technical hassles.
